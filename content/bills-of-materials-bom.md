## Components

Every item included also had a set of sub-items it came with. For example, each servo came with horns included that we used to attach the 3D-printed parts onto. For some, we used some if not all parts included with to attach properly.

![204_1_Servo_Horns_and_Hardware.jpg](Bills%20of%20Materials%20(BOM)/204_1_Servo_Horns_and_Hardware.jpg)

Another example of this is with the Raspberry Pi 4 (RPi4). Included comes the power supply that is needed to power the RPi itself, but it is necessary when using.
